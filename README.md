
## Info
Web application which enables customers to buy or customize jewelry.

## Features 

### User Managment
* Registration and login

### Product Managment
* Add a product
* List products
* Edit a product
* Delete a product

### Customization 
* Engraving
* Personalized jewelry

### Shopping cart
* Create a cart
* Remove cart
* See cart
* Add item
* Remove item
* Place order

### Searching and sorting
* Search products by name, categories, materials..
* Filter search by lower/higher price..
* Sorting by price, popularity...

## Technologies
* Java
* Spring boot
* Thymeleaf + bootstrap
* SQL
